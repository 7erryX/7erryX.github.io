    Tip1 - 整数比较的坑

    在 Python 中一切都是对象，整数也是对象，在比较两个整数时有两个运算符 == 和 is，它们的区别是：

    • is 比较的是两个整数对象的 id 值是否相等，也就是比较两个引用是否代表了内存中同一个地址。
    • == 比较的是两个整数对象的内容是否相等，使用 == 时其实是调用了对象的 __eq__() 方法。
